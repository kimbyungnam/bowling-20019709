# Plan

## 목표
한 프레임에서 스페어(두 투구 합이 10)를 기록하면, 해당 프레임 점수에 다음 투구 1회의 핀 수를 보너스로 더한다.

## 실패하는 테스트
`tests/test_game.py::test_one_spare_adds_next_roll_as_bonus`
- 투구: 5, 5(스페어), 3, 이후 17번 거터볼(0)
- 기대 점수: 16 (frame1 = 5+5+3 보너스 = 13, frame2 = 3+0 = 3, 나머지 0)
- 현재는 단순 합계(13)를 반환해 실패한다.

## 접근
- `score()`를 프레임 단위로 순회하는 방식으로 재작성한다.
- 각 프레임에서 두 투구의 합이 10이면(스페어) 해당 프레임 점수 = 두 투구 합(10) + 다음 투구 1회의 핀 수.
- 스페어가 아닌 일반 프레임은 두 투구의 합을 그대로 프레임 점수로 사용한다.
- 스트라이크와 10번 프레임의 특수 규칙은 이번 사이클 범위에 포함하지 않고, 이후 사이클에서 각각 다룬다.
