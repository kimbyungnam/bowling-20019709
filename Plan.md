# Plan

## 목표
10번 프레임 특수 규칙(스트라이크/스페어 시 추가 투구 허용)을 포함해 퍼펙트 게임(12회 연속 스트라이크)이 300점으로 계산되는지 확인한다.

## 결과
`tests/test_game.py::test_perfect_game_scores_300`을 작성해 실행한 결과, 별도의 프로덕션 코드 변경 없이 즉시 통과했다.

기존 `score()` 구현이 "스트라이크/스페어면 다음 1~2개 투구를 보너스로 더한다"는 규칙을 프레임 번호와 무관하게 동일하게 적용하는데, 10번 프레임의 추가 투구 허용 규칙이 정확히 그 보너스에 필요한 투구들을 제공하기 때문에 별도의 10번 프레임 분기 없이도 일반화되어 동작한다. 손으로 추가 검증한 케이스(8·9·10번 프레임 연속 스트라이크, 10번 프레임 스페어 등)도 모두 올바르게 계산됨을 확인했다.

## 처리
새 프로덕션 코드는 작성하지 않는다. 이 테스트는 이미 구현된 동작(10번 프레임 규칙)을 검증하고 문서화하는 회귀 테스트로서 커밋한다.
